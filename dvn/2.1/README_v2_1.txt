This README file contains information about the files needed
to install the Dataverse Network (DVN) software version 2.1.

 
Files needed for Fresh installation:

- DVN-EAR-v2_1.ear 

- config_v2_1.zip, config files to be put into the glassfish
  domain directory 

- referenceData_v2_1.sql

Follow instructions for a fresh installation from http://thedata.org/book/installers-guide)

______________________________________________________________________________

Files needed for an upgrade from v2.0 to 2.1:

- DVN-EAR-2-1.ear 

- jhove.conf and oaicat.properties, contained in config_v2_1.zip  

- buildupdate_v2_0_v2_1.sql

______________________________________________________________________________

For instructions on upgrading from DVN 2.0 to DVN 2.1, please visit:
http://thedata.org/blog/how-upgrade-dvn-v20-v21
