Welcome to the long-awaited release of the DVN app version 3.0!
==============================================================

For a list and description of new features in this release, please consult 

http://thedata.org/pages/latest-release

One of the important new features is the new API for accessing DVN data. For technical details, please consult this document: 
https://sourceforge.net/projects/dvn/files/dvn/3.0/dvnapi_v1_0.pdf

More documentation is available at http://guides.thedata.org/. 


Installing the DVN.
================== 

If this is a new installation:
-----------------------------

A scripted installer is provided. Please consult the Installers Guide at

http://projects.iq.harvard.edu/thedatanew_guides/book/installers-guides.

The installer package itself can be found here:

https://sourceforge.net/projects/dvn/files/dvn/3.0/dvninstall_v3_0.zip.

If you are upgrading an existing DVN v.2.2.5 installation:
---------------------------------------------------------

Please consult the upgrade instructions available here on Sourceforge: 

https://sourceforge.net/projects/dvn/files/dvn/3.0/dvnupgrade_v3_0.pdf
 
(And if you are upgrading from a version of the DVN *earlier* than 2.2.5, 
please also consult the README files for *all* the versions in between!)

---

As always, if you have any questions and/or run into problems, please
contact us at dvn_support@help.hmdc.harvard.edu.
